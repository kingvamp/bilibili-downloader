# 📺 Bilibili Downloader

一个基于 Electron + [BBDown](https://github.com/nilaoda/BBDown) 构建的现代化 Bilibili 视频下载器。拥有极简的 UI 设计、智能的链接解析，以及强大的后台剪贴板静默下载能力。

本软件专为原生桌面体验打造，并深度适配了浏览器增强插件 **[Bilibili Enhancer](https://github.com/kingvamp/bilibili-enhancer)**，为您带来“网页端一键触发，本地全自动静默下载”的终极无缝闭环体验。

## ✨ 核心特性

- **📱 扫码登录**：支持 B 站客户端扫码登录，自动保存 Cookie，解锁 1080P 高画质及会员专享画质。
- **🤖 智能链接解析**：只需一个输入框。无论粘贴单视频 (`BV...`)、分P教程、收藏夹链接，还是纯数字播单 ID，均能自动识别并应用最优下载策略。
- **📝 双轨字幕封装**：原生支持同时下载 UP 主手工字幕与 AI 智能字幕，并自动作为独立的软字幕轨道封装进 `.mp4` 文件中，播放时可自由切换。
- **📂 完美的多P命名**：彻底解决批量下载或多P视频命名错乱问题，采用规范的 `主标题 - P01 分P标题 [BV号]` 格式。
- **📥 后台剪贴板唤醒**：开启后，复制常规 B 站链接会自动唤醒主窗口并自动填入，随时待命。

## 🚀 梦幻联动：Bilibili Enhancer 全自动静默下载

本项目独家实现了基于系统剪贴板的轻量级 **IPC 跨进程通信**，完美联动浏览器插件 **[Bilibili Enhancer](https://github.com/kingvamp/bilibili-enhancer)**：

当你在 Bilibili Enhancer 插件中触发下载时,本地下载器直接在后台静默建立下载队列。下载成功结束后，浏览器插件会在网页端向用户展示“下载完成”的状态更新。

> 真正实现了从浏览器到本地磁盘的“零感”下载工作流！

## ⚙️ 偏好设置

点击主界面右上角的 `⚙️` 图标可进入偏好设置（所有设置均支持本地持久化保存）：
- 📂 **自定义路径**：更改默认下载保存目录。
- 📋 **智能唤醒**：开启/关闭复制普通 B 站链接时自动唤醒窗口。
- 📝 **字幕封装**：开启/关闭双轨字幕（人工+AI）的强制下载与封装功能。
